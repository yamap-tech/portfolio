# My Portfolio

## 概要
このアプリは [課題] を解決するための Web アプリケーションです。  
Java（Spring Boot）を使用し、バックエンド開発の基礎力を示すことを目的としています。

## 技術スタック　予定
- Java 17
- Spring Boot 3.x
- Spring Data JPA
- H2 Database（開発用） / PostgreSQL（本番想定）
- JUnit 5 / Mockito
- Docker / Docker Compose
- GitHub Actions（CI）

## 機能一覧（予定）
- [ ] REST API（CRUD）
- [ ] 永続化（JPA）
- [ ] バリデーション & 例外処理
- [ ] 単体テスト / 統合テスト
- [ ] Docker化
- [ ] CI（GitHub Actions）
